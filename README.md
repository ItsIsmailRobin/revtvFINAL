## 🚀 Quick Start

```bash
# Clone the repository
git clone https://github.com/ItsIsmailRobin/RevTV.git

# Navigate to directory
cd RevTV

# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build
```
