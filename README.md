## 🚀 Quick Start

```bash
# Clone the repository
git clone https://github.com/ItsIsmailRobin/RevTV.git

# Navigate to directory
cd RevTV

# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build
```

## ⌨️ Keyboard Shortcuts

| Key | Action |
|:---:|:---|
| `Space` | Play / Pause |
| `F` | Toggle Fullscreen |
| `M` | Mute / Unmute |
| `↑` `↓` | Volume Up / Down |
| `←` `→` | Seek -10s / +10s |
